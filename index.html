Software Dev

Tixel Bot


Tixel Bot is a useful assistant to find second-hand tickets on the ticket resale website Tixel. By employing a custom web scraper and a messaging bot, Tixel Bot efficiently scours tickets for desired events and promptly notifies users when a ticket falls below their specified price threshold.

The project was conceived to tackle the challenges associated with finding second-hand event tickets, enhancing the conventional email notification system by introducing an easier and user-centric messenger bot interface.

Using a scrapy, a web scraping framework for python, python telegram bot framework for telegram bots and deploy with docker, this project provided significant exposure to web technologies and emphasized a design approach that prioritised the user experience. The targeted user group for this bot included some close friends and myself.

For an in-depth understanding of this project, a blog post detailing the journey to project completion is available here.

Note: This bot is intended for personal use only and is not intended for commercial purposes, thus remaining inaccessible to the public. The source code can be found on Github.
